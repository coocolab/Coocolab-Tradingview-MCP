# 技能：策略報告

## 說明
讀取 TradingView 策略測試器的結果，產出結構化的績效報告。

## 使用時機
- 使用者說「分析我的策略績效」、「策略測試結果怎麼樣」、「幫我看回測報告」
- 需要系統化評估一個交易策略

## 執行步驟

1. `ui_open_panel (panel: "strategy-tester")` → 開啟策略測試器面板
2. `capture_screenshot (region: "strategy_tester")` → 截取策略測試器畫面
3. `data_get_pine_tables (study_filter: "策略名稱")` → 讀取策略統計資料表

## 輸出格式

```
【策略績效報告】

策略名稱：EMA 交叉策略
測試期間：2024-01-01 ~ 2025-01-01
標的：BTCUSD | 時間框架：4H

【核心指標】
- 淨損益：+$12,450（+24.9%）
- 勝率：58.3%（105/180 筆）
- 盈虧比：1.8:1
- 最大回撤：-15.2%
- Sharpe Ratio：1.34

【交易統計】
- 總交易次數：180 筆
- 平均持倉時間：2.3 天
- 最大連勝：7 筆
- 最大連敗：5 筆

【評估】
勝率和盈虧比都在合理範圍，最大回撤需要注意。
建議加入停損規則降低回撤。
```

## 注意事項
- 策略必須已加到圖表上並執行完回測
- `data_get_pine_tables` 的 `study_filter` 要對應策略的確切名稱
- 截圖通常比讀資料更快取得策略測試器的完整結果
