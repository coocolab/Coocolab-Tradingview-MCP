# 貢獻指南

感謝你願意貢獻！這個專案改寫自 [@tradesdontlie](https://github.com/tradesdontlie) 的原版工具，歡迎任何改善。

## 怎麼貢獻

1. Fork 這個 repo
2. 建立你的功能分支：`git checkout -b feature/你的功能名稱`
3. Commit 你的修改：`git commit -m '新增：你的功能說明'`
4. Push 到分支：`git push origin feature/你的功能名稱`
5. 開一個 Pull Request

## 回報問題

遇到 bug 或有功能建議，請直接開 Issue，並附上：
- 你的作業系統
- Node.js 版本
- TradingView 版本
- 錯誤訊息或重現步驟

## 程式碼規範

- 保持跟原版一致的程式碼風格
- 新工具請加對應的說明到 `CLAUDE.md`
- 如果改動影響安裝流程，記得更新 `SETUP_GUIDE.md`

## 關於原版專案

這個版本的改動不會自動同步回 [@tradesdontlie](https://github.com/tradesdontlie/tradingview-mcp) 的原版。如果你發現原版的 bug，建議也在原版 repo 回報。
